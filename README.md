# UK-Train-Rides-2023-24
This repository houses the documentation and deliverables for a data analysis project exploring UK train journey data from 2023 to 2024.
The project aims to explore purchasing patterns, journey statistics, and factors influencing delays and refund requests within the provided dataset.

OBJECTIVES:
1.Key Performance Indicators
2.Sales and Financial Analysis
3.Operational and Journey Analysis

METHODOLOGY AND TOOLS:
1.Data Cleaning: Performed manually and using functions within Microsoft Excel to handle missing values, correct data types, and ensure consistency.
2.Data Analysis & Visualization: The cleaned data was imported into Power BI for creating measures, relationships, and the final interactive report.
